# 🎯 Number Guessing Game

A simple **Java Console Application** developed as **Project 1** during my **Decode Labs Java Programming Internship**.

The game generates a random number between **1 and 100**, and the player must guess it within a limited number of attempts. After each guess, the program provides feedback indicating whether the guess is too high or too low, making the game interactive and engaging.

---

## 📌 Features

- 🎲 Random number generation (1–100)
- ⌨️ User input using `Scanner`
- 📈 High/Low guess feedback
- 🎯 Limited number of attempts
- 🏆 Score tracking
- 🔄 Multiple game rounds
- ✅ Input validation
- ⚠️ Exception handling for invalid inputs

## ▶️ Getting Started

### Prerequisites

- Java JDK 8 or later
- IntelliJ IDEA (or any Java IDE)

### Clone the Repository

```bash
git clone https://github.com/kapoormayank9696/Project-1-Number-Guessing-Game.git
```

### Navigate to the Project

```bash
cd Project-1-Number-Guessing-Game
```

### Compile the Program

```bash
javac src/DecodeLabs_Java_P1.java
```

### Run the Program

```bash
java -cp src DecodeLabs_Java_P1
```

## 🎮 How to Play

1. Run the application.
2. The program generates a random number between **1 and 100**.
3. Enter your guess.
4. Receive feedback:
   - 📉 Too Low
   - 📈 Too High
   - ✅ Correct Guess
5. Try to guess the number within the allowed attempts.
6. Play multiple rounds and improve your score.
